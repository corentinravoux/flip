Likelihoods
===========

Velocity alone
--------------

Standard velocity likelihood:

.. math::
    
    `\mathcal{L} = (2\pi)^\frac{N}{2}|C|^\frac{1}{2} \exp\left[\frac{1}{2} \mathbf{v}^T C^{-1}\mathbf{v}\right]` 

where :math:`C` is such that:

.. math::

    `C = C_\mathrm{vv} + \sigma_v^2 \mathbb{I}_n`
